# Cursor Mobile Demo

用于验证从手机端（Cursor for iOS / [cursor.com/agents](https://cursor.com/agents)）通过 Cloud Agents 操作本仓库。

## 手机端怎么用

1. 在 [Cursor Dashboard](https://cursor.com/dashboard) 连接 GitHub，并授权本仓库。
2. iPhone：安装 [Cursor App](https://apps.apple.com/app/cursor/id6767085653)，登录同一账号。
3. Android / 通用：打开 [cursor.com/agents](https://cursor.com/agents)。
4. 选择本仓库与分支，发送任务（例如：「把标题改成 Hello from phone」）。
5. 在手机上查看 diff，确认后合并 PR。

需要 Cursor 付费计划（Pro 及以上）且开启 Cloud Agents。
